# Gash
### by [Elvin Yung](https://github.com/elvinyung)

Gash is a simple Go hashmap library, containing a few implementations of various types of hashmaps. 

Checklist:
* [x] chain (simple) hash
* [x] closed hash
* [x] cuckoo hash
* [ ] double hash
* [ ] coalesced hash
* [ ] some sort of hash set
* [ ] some sort of bloom filter
